# TASK-MANAGEMENT 📋

A task management application built with JavaScript for organizing and tracking tasks efficiently.

## Overview

TASK-MANAGEMENT is a web-based application designed to help users create, organize, and manage their tasks effectively. It provides a simple and intuitive interface for task tracking and project management.

## Features

- ✅ **Create Tasks** - Easily create and add new tasks to your list
- 📝 **Edit Tasks** - Modify task details and descriptions
- ✔️ **Mark Complete** - Check off completed tasks
- 🗑️ **Delete Tasks** - Remove tasks from your list
- 🏷️ **Organize** - Categorize and organize tasks by priority or status
- 🔍 **Search & Filter** - Find tasks quickly with search functionality
- 📊 **Task Dashboard** - Overview of all your tasks at a glance

## Tech Stack

- **Language**: JavaScript
- **Frontend**: HTML, CSS, JavaScript

## Usage

1. **Create a Task** - Click the "Add Task" button and enter task details
2. **Edit a Task** - Click on a task to edit its title, description, or priority
3. **Complete a Task** - Check the checkbox next to a task to mark it as completed
4. **Delete a Task** - Click the delete button to remove a task
5. **Filter Tasks** - Use the filter options to view tasks by status or priority
6. **Search Tasks** - Use the search bar to find specific tasks
